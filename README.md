# 🌦️ Django Weather App

A simple and responsive Weather Application built with **Django** that allows users to search for any city and get real-time weather information using a Weather API.

---

## Direct link: https://django-weather-app-blond.vercel.app/

## 📌 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- ☁️ Shows weather condition
- 💧 Humidity information
- 🌬️ Wind speed
- 📍 Easy-to-use interface
- 📱 Responsive design

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/SudarshanAg31/django_weather_app.git
```

### 2. Navigate to project

```bash
cd django_weather_app
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

### 6. Run Migrations

```bash
python manage.py migrate
```

### 7. Start Development Server

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
